# Elasticsearch Client

A simple Elasticsearch client, to make use of queries in C++ code.

## Prerequisites:
To get the best results note the following:
* C++ version >= 17
* CMake version >= 3.22
* Elasticsearch version >= 7.5

## Dependencies:
* libcurl version >= 7.17.0
